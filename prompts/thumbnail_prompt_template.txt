## IMAGE PROMPT RULES FOR NANO BANANA PRO / MIDJOURNEY
## ===============================================================

**STEP 1: ANALYZE & COUNT PEOPLE**
- 1 person → SINGLE FACE layout
- 2 people → SPLIT SCREEN layout
- 3+ people → MULTI PANEL layout

**STEP 2: DETECT EMOTION FROM CONTEXT**
- Injury/crash/hospital → pain, suffering, bandages
- Drama/beef/callout → angry vs scared
- Arrested/jail/FBI → crying, defeated, handcuffs
- Retired/quit/finished → sad, emotional, tears
- Reaction/response → shocked, screaming

**STEP 3: CHOOSE LAYOUT**

【1 PERSON - SINGLE FACE】
- That person fills 80% of frame
- Extreme close-up face
- Emotion matches context
- Inset image showing incident (small, corner)
- Background: blurred hospital/jail/press conference

【2 PEOPLE - SPLIT SCREEN】
- Split screen 50/50
- LEFT = attacker/reactor (angry, yelling)
- RIGHT = victim/target (crying, defeated)
- Center inset with red border + red arrow

【3+ PEOPLE - MULTI PANEL】
- Main person large on left (40%)
- Others smaller on right (collage)

**STEP 4: THUMBNAIL TEXT STYLE**
- Quotation marks, ALL CAPS
- Giant bold BLACK text, thick WHITE outline
- Bottom 30% of thumbnail
- 2-4 word shocking quote
- Censor curse words (F*CK, SH*T, B*TCH)

**STEP 5: REQUIRED ELEMENTS**
- "BREAKING" or "EXCLUSIVE" red label bottom left (always)
- "LIVE" badge if breaking news
- Red arrow pointing to key detail
- CNN-style news look for serious topics

**STEP 6: REALISM REQUIREMENTS (CRITICAL - MAKE IT LOOK REAL, NOT AI)**
- Use "professional sports photography" or "press conference photo" style
- Specify "shot on iPhone" or "shot on Canon DSLR" for natural look
- Add "natural skin texture", "pores visible", "slight imperfections"
- Include "authentic lighting", "real stadium lights" or "natural sunlight"
- Say "photojournalism style" NOT "hyper-detailed CGI"
- Avoid words like: "perfect", "flawless", "digital art", "rendered"
- Use: "candid moment", "raw emotion", "unedited", "documentary style"

**STEP 6.5: USE REAL NAMES (CRITICAL FOR RECOGNITION)**
- ALWAYS use the person's ACTUAL NAME, never generic terms
- ❌ DON'T SAY: "WNBA player", "basketball player", "athlete", "coach", "fighter"
- ✅ DO SAY: "Caitlin Clark", "Angel Reese", "A'ja Wilson", "Paige Bueckers"
- Include identifying details: "Caitlin Clark in Indiana Fever #22 jersey"
- For multiple people, name EACH person specifically
- Example: "Caitlin Clark (left) and Angel Reese (right)"
- Add team/uniform details: "wearing Iowa Hawkeyes uniform", "in UFC octagon"

**STEP 6.6: REFERENCE IMAGE INSTRUCTIONS (MANDATORY)**
When user attaches a reference image, the AI MUST use it. The image_prompt must ALWAYS start with a STRONG mandatory instruction.

ALWAYS start the image_prompt with this EXACT instruction block:
"IMPORTANT: You MUST use the attached reference image. Extract and keep the EXACT faces, skin tones, facial features, and any logos/screenshots/articles shown in the reference. Do NOT generate new faces - use the faces FROM the reference image. Apply more dramatic expressions to those same faces, improve composition, but the people must look IDENTICAL to the reference."

This ensures:
- AI is FORCED to use the reference image (not optional)
- Faces match the original thumbnail
- Only expressions and composition are enhanced

**STEP 7: EMOTION DETAILS (MAKE FACES BELIEVABLE)**
- EXTREME emotions but NATURAL execution
- Eyes WIDE OPEN, looking at camera (real surprise, not cartoon)
- Mouth OPEN (authentic screaming/shock, not exaggerated)
- Real tears, real sweat (not glossy or fake)
- Natural wrinkles when screaming (forehead lines, crow's feet)
- Skin reacts naturally (red face when angry, pale when shocked)
