Setup & API Configuration

Required APIs & Services

1. Google Gemini API Free Tier Available

Used for: AI script rewriting, thumbnail ideas generation

  • Go to: https://aistudio.google.com/apikey
  • Sign in with Google account
  • Click "Create API Key"
  • Copy the API key to Settings > API Keys
Free Tier: 60 requests/minute, 1500 requests/day - enough for most users!

2. Claude API (Optional) Paid

Used for: Alternative AI script rewriting (higher quality)

  • Go to: https://console.anthropic.com/
  • Create account and add payment method
  • Go to API Keys > Create Key
  • Copy to Settings > API Keys
Pricing: ~$3 per 1M input tokens, ~$15 per 1M output tokens

3. Fish Audio (Voice Generation) Paid for API

Used for: AI voiceover generation with realistic voices

Option A: Browser Method (Recommended - Uses Your Subscription)
  • Go to: https://fish.audio/
  • Create account and subscribe to a plan
  • The software will open browser and use your logged-in session
  • Login once, cookies are saved for future use
Option B: API Method (Direct API calls)
  • Go to Fish Audio dashboard
  • Navigate to API section
  • Generate API key
  • Add to Settings > API Keys > Fish Audio API Key
Fish Audio Plans:
- Free: Limited characters/month
- Basic: ~$10/month - 100,000 characters
- Pro: ~$30/month - 500,000 characters
- Enterprise: Custom pricing
Tip: Browser method uses your subscription directly - no separate API costs!

4. YouTube Channel Setup

Used for: Automatic video upload

  • Each profile can have its own YouTube channel
  • First upload will ask you to login in browser
  • Cookies are saved per channel profile
  • Videos upload as UNLISTED (you publish manually)
Important: Login is saved in browser profile. Don't delete browser_profiles folder!

Profiles & Channels

Profiles allow you to create multiple versions of the same video with different styles, voices, and target channels.

Profile Configuration

Setting Description
name Display name for the profile
suffix Short code used in folder names (e.g., BASKLY, FASTIQ)
prompt_file AI prompt template for script rewriting
default_voice Fish Audio voice model name
background_video Background video for styling
background_music Music file for final video
youtube_channel Target YouTube channel name
enable_upload Auto-upload to YouTube (true/false)
upload_wait_minutes Minutes to wait for upload completion

Example Profile

{
  "SPORTS": {
    "name": "Sports Channel",
    "description": "High-energy sports content",
    "suffix": "SPORTS",
    "prompt_file": "./prompts/sports_prompt.txt",
    "default_voice": "energetic_male",
    "background_video": "./backgrounds/sports_bg.mp4",
    "background_music": "./music/upbeat.mp3",
    "youtube_channel": "MySportsChannel",
    "enable_upload": true,
    "upload_wait_minutes": 5
  }
}

Multi-Channel Workflow

  1. Create profiles for each channel (different prompts, voices, styles)
  2. Run "Create Video" or "Recreate Video"
  3. Select multiple profiles when asked
  4. Software generates separate videos for each profile
  5. Each video uploads to its own YouTube channel

Troubleshooting

Fish Audio Login Issues

Problem: Browser opens but not logged in

Solution:

  1. Delete browser_profiles folder
  2. Run voiceover step again
  3. Login when browser opens
  4. Cookies will be saved for next time

YouTube Upload Fails

Problem: Upload doesn't complete or browser closes

Solution:

  1. Increase upload_wait_minutes in profile settings
  2. Check internet connection stability
  3. Make sure video file isn't corrupted

GPU Encoding Errors

Problem: NVENC errors or encoding failures

Solution:

  1. Update NVIDIA drivers
  2. Disable GPU encoding in settings (falls back to CPU)
  3. Check if another app is using GPU encoder

Whisper Transcription Slow

Problem: Step 3 takes very long time

Solution:

  1. Use smaller model (small instead of large)
  2. Enable GPU if available
  3. Use faster-whisper instead of standard whisper

API Rate Limits

Problem: AI script generation fails with rate limit error

Solution:

  1. Wait a few minutes and retry
  2. Switch to different AI provider (Gemini ↔ Claude)
  3. Check API quota in provider dashboard
